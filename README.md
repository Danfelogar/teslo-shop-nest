# TesloDB API

1. clone project
2. ```yarn install```
3. clone the file ```.env.template``` and rename to ```.env```
4. change environment
5. rise up data base
```
docker-compose up -d
```
6. SEED execution
```
localhost:3000/api/seed
```
7. run project ```yarn start:dev```
